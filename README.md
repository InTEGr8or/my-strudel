# My Strudel Site

A minimal Strudel song manager built with [11ty](https://www.11ty.dev/).

## Usage

1.  **Start development server**: `npm start`
2.  **Build static site**: `npm run build`
3.  **Add songs**: Drop `.strudel` or `.tidal` files into the `src/songs/` directory.

## Features

- **Automatic Metadata**: Uses `@title` and other tags from your Strudel files.
- **Embedded REPL**: Each song page has a functional Strudel REPL.
- **Minimalist**: No heavy frameworks, just standard web components.
